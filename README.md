# Gerador de Dados Fakes (Fakedata)

Este projeto contém scripts Python para gerar bases de dados com características realistas (clientes, vendas, itens_venda, produtos, vendedor) para fins de teste, prototipagem ou análise.

## Configuração e Execução

### Pré-requisitos
Certifique-se de ter o Python 3 instalado.

### 1. Clonar o Repositório

Para obter uma cópia completa do projeto para sua máquina, use o comando `git clone` e navegue para o diretório:

```bash
git clone [https://github.com/marciohcsouza/Fakedata.git](https://github.com/marciohcsouza/Fakedata.git)
cd Fakedata
